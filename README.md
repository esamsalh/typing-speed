
## npm Package

Install the reusable typing-speed calculation library:

```bash
npm install @toolrar/typing-speed-core

أداة اختبار سرعة الكتابة على الكيبورد ⌨️

مشروع برمجى مفتوح المصدر ومصمم خصيصاً لتوفير **أداة اختبار سرعة الكتابة على الكيبورد** باللغة العربية بشكل دقيق وفوري. يهدف المشروع إلى مساعدة المستخدمين والمدخلين البيانات في قياس كفاءتهم وتطوير مهارات الكتابة السريعة عبر لوحة المفاتيح دون أي تعقيد.

## ✨ مميزات المشروع
* **قياس دقيق ولحظي:** فحص سرعة الكتابة بناءً على معيار الكلمة في الدقيقة (WPM).
* **حساب نسبة الدقة:** تتبع الأخطاء الإملائية مباشرة أثناء الكتابة وحساب النسبة المئوية للاتزان والدقة.
* **عداد تنازلي ذكي:** يبدأ المؤقت التلقائي بالعمل فور لمس أول حرف على الكيبورد.
* **متوافق مع محركات البحث:** مهيأ كصفحة ويب سريعة التحميل وخفيفة الحجم لتقديم أفضل تجربة مستخدم.

## 🚀 التشغيل المباشر والأدوات المتقدمة
إذا كنت تبحث عن اختبار احترافي وشامل مجهز بإحصائيات متقدمة وخيارات متعددة للنصوص العربية والإنجليزية، نقترح عليك تجربة الأدوات المتاحة أونلاين:
1. يمكنك الانتقال مباشرة لإجراء الفحص الشامل عبر: [أداة اختبار سرعة الكتابة على الكيبورد المتقدمة](https://www.toolrar.com/General/typing-speed)
2. لمتابعة المزيد من أدوات الويب المجانية والتطبيقات الذكية، قم بزيارة منصتنا الرئيسية: [موقع ToolRar الرسمي](https://www.toolrar.com)
